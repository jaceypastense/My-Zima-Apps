## List of Apps

[Backrest](https://github.com/garethgeorge/backrest) 

Backrest is a web-accessible backup solution built on top of restic

[Metube](https://github.com/alexta69/metube)

Web GUI for youtube-dl (using the yt-dlp fork) with playlist support. Allows you to download videos from YouTube and dozens of other sites.

[Newt](https://github.com/fosrl/newt)

Newt is a fully user space WireGuard tunnel client and TCP/UDP proxy, designed to securely expose private resources controlled by Pangolin. By using Newt, you don't need to manage complex WireGuard tunnels and NATing.

[Tubesync](https://github.com/meeb/tubesync)

TubeSync is a PVR (personal video recorder) for YouTube. Or, like Sonarr but for YouTube (with a built-in download client). It is designed to synchronize channels and playlists from YouTube to local directories and update your media server once media is downloaded.

[MediaCMS](https://github.com/mediacms-io/mediacms) - under test

MediaCMS is a modern, fully featured open source video and media CMS. It is developed to meet the needs of modern web platforms for viewing and sharing media. It can be used to build a small to medium video and media portal within minutes.
